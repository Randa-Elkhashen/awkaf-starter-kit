## IMPORTANT RULES

[Naming-Convention]

# [Classes/Enums] 
=> XController, XModel, e.g. UserController
  [enum-values] => X_VALUE  e.g USER_VALUE

# [Files/Folders] 
=> use _, x_controller, x_model, .. e.g. user_controller

# [Variables/Functions]
=> camelCase, e.g _user, _getUserData

## [Best-practice] ## 
=> Don't Use Abbreviations
